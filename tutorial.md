# Initialise code
`git init`

# Create new file
`touch <file name>`

# Create directory 
`mkdir <Dir Name>`

# Check status of the code
`git status`

# To track files
`git add <folder name> <filename>`

# To add commit message
`git commit -m "your message"`

# In order to push changes to remote repository(cloud)
`git push origi main`
